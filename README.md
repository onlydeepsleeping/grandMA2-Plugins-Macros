# grandMA2-Plugins-Macros-public
current versions:
At Color P6 v1  (finished release)
  - can apply internal color macros of Cameo P6 via user input, plugin for MA2 lighting consoles

At Color v0.4   (merged At Color P6 v1 into plugin, up to date version)
  - can apply colors of pool to selected fixtures via user input (merged: Color P6), plugin for MA2 lighting consoles

XML-Lua-Creator-v1.exe  (standalone .exe for win x64 systems)
  - can generate .xml files for .lua files, needed for inport of plugins into MA2 lighting consoles
  - supports input and output paths, drag and drop, copying created files to clipboard

Create-GrandMA2-XML.ps1
  - previous version of XML-Lua-Creator-v1.exe, code is visible by editing the file (unlike the .exe)
